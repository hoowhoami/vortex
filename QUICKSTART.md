# Vortex 快速启动指南

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run tauri:dev
```

### 生产构建
```bash
npm run tauri:build
```

## 🎯 核心功能

### 1. 连接管理
- **添加连接**: 点击侧边栏的 `+` 按钮
- **分组管理**: 创建分组来组织连接
- **快速连接**: 右键菜单选择 "Connect"
- **编辑/删除**: 右键菜单操作

### 2. 键浏览
- **搜索键**: 使用模式匹配（如 `user:*`）
- **虚拟滚动**: 支持 100,000+ 键无卡顿
- **批量操作**:
  1. 点击 "Bulk Select" 进入批量模式
  2. 选择需要的键
  3. 点击删除按钮批量删除

### 3. 值编辑
- **支持类型**: String, Hash, List, Set, ZSet
- **JSON 格式化**: 自动识别 JSON 并提供格式化按钮
- **TTL 管理**: 点击 TTL 徽章编辑过期时间
- **实时保存**: 修改后点击保存按钮

### 4. 命令面板
按 `Cmd/Ctrl + K` 打开，支持：
- 刷新连接列表
- 刷新键列表
- 快速跳转到连接
- 搜索最近访问的键

## 💡 使用技巧

### 高效操作
1. **使用命令面板**: `Cmd/Ctrl + K` 是最快的操作方式
2. **批量删除**: 需要删除多个键时，使用批量模式
3. **JSON 格式化**: 处理 JSON 数据时，一键格式化提升可读性
4. **分组管理**: 使用分组来组织不同环境的连接

### 性能优化
- 虚拟滚动自动处理大量键
- 搜索时使用精确的模式减少结果数量
- 定期清理不需要的连接

### 快捷操作
- 双击连接快速连接
- 右键菜单快速访问常用功能
- 使用搜索框快速定位键

## 🔧 配置说明

### 连接配置
- **Host**: Redis 服务器地址（默认 localhost）
- **Port**: Redis 端口（默认 6379）
- **Password**: 密码（可选）
- **Database**: 数据库编号（默认 0）
- **SSL**: 是否使用 SSL 连接

### 应用设置
- **主题**: 浅色/深色/跟随系统
- **语言**: 中文/English
- **其他**: 更多设置即将推出

## 🐛 常见问题

### Q: 连接失败怎么办？
A: 检查以下几点：
1. Redis 服务是否运行
2. 主机和端口是否正确
3. 密码是否正确
4. 防火墙是否允许连接

### Q: 键列表加载很慢？
A: 使用更精确的搜索模式，避免使用 `*` 匹配所有键

### Q: 如何导出连接配置？
A: 此功能即将推出

### Q: 支持哪些 Redis 版本？
A: 支持 Redis 2.8 及以上版本
