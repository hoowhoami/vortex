# EchoMusic

一个基于 Vue 3 的现代音乐播放器应用。

## 技术栈

- Vue 3 + TypeScript
- Vite
- Naive UI
- Pinia
- Vue Router
- Tailwind CSS
- Howler.js

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build

# 预览构建
npm run preview

# 代码检查
npm run lint

# 代码格式化
npm run format
```
